Базовые примеры работы компонентов Ability можно найти в папке Scenes.

Компоненты упрощены, они демонстрируют только общий принцип работы модульных способностей.

Для создания новой способности нужно выбрать Create - new Ablity. После этого появится новый ScriptableObject с полями:

 - Ability Name - поле, где мы можем указать отображаемое в игре название способности

 - Diration, где мы можем в секундах указать общее для всех подкомпонентов время работы

 - Component Configs - массив подкомпонентов, каждый из которых сработает при активации способности


Типы подкомпонентов:

- PlayAnimation - запускает анимационный клип по названию содержащего его состояния в анимационном контроллере

- PlaySound - запускает аудио клип, имеет для этого поле выбора

- AreaDamage - подкомпонент для нанесения урона по области, создает сферу с центром в авктивирующем способность персонаже с указанным радиусом, уроном

  и количеством коллайдеров, к которым будет применена проверка на возможность нанесения им урона. Также с помощью поля RotateToFirstTarget - позволяет персонажу развернуться к первой обнаруженной цели.
 
 Поле ResponseAbility - ссылается на ответную способность, которая будет применена каждым получившим урон объектом.

- PlayParticle - подкомпонент, имеющий поле выбора префаба объекта Particle (появляющегося при активации способности и уничтожающегося в ее конце)

- Move - подкомпонент, позволяющий персонажу передвигатсья во время действия способности по четырем направлениям и с заданной скоростью

- Buff - подкомпонент, отвечающий за усиления параметров персонажа в момент проигрывания способности
